Cypress OrangeHRM
--
Projeto de automação de testes End-to-End (E2E) utilizando Cypress no sistema OrangeHRM.

O objetivo deste repositório é praticar automação de testes web, validação de fluxos críticos e aplicação de boas práticas de QA em um ambiente próximo do cenário real.
----
📌 Objetivos do Projeto
Automatizar fluxos críticos do OrangeHRM
Praticar testes End-to-End com Cypress
Estruturar testes de forma organizada e escalável
Aplicar conceitos de QA em cenários reais
Evoluir futuramente para boas práticas de automação profissional

⚙️ Como Executar o Projeto
1. Clone o repositório
git clone https://github.com/wlidemberg/cypress-orangehrm.git
2. Acesse a pasta do projeto
cd cypress-orangehrm
3. Instale as dependências
npm install
4. Execute o Cypress

Modo interface gráfica:

npx cypress open

Modo headless:

npx cypress run
